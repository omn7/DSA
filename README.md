## How to Run

1. Compile the Java file:
   ```bash
   javac filename.java
   ```

2. Run the compiled class:
   ```bash
   java filename
   ```

## File Structure

- `.java` files contain the source code
- `.class` files are the compiled bytecode
- Each problem has its own solution file
